## 安装

```bash
# 通过 npm 安装
npm i sixty -S
```

## 快速上手

```js
import Vue from "vue";
import { Button } from "sixty";
import "sixty/lib/index.css";

Vue.use(Button);
```

sixty 也支持按需引入

## 浏览器支持

现代浏览器以及 Android 4.0+, iOS 7+.
